# Awesome Scientific Illustration [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of tools, resources, and tutorials for creating scientific illustrations, figures, and graphics for research papers, journals, and presentations.

Whether you're a PhD student, postdoc, or research scientist, this list will help you create publication-ready figures without spending weeks learning design software.

## Data Visualization

### Python Libraries

```python
# Essential libraries for scientific figures
pip install matplotlib seaborn plotly bokeh altair
```

- [Matplotlib](https://matplotlib.org) - The classic Python plotting library
- [Seaborn](https://seaborn.pydata.org) - Statistical data visualization
- [Plotly](https://plotly.com/python/) - Interactive plots
- [Bokeh](https://bokeh.org) - Interactive visualization library
- [Altair](https://altair-viz.github.io) - Declarative statistical visualization

### R Libraries

```r
# Essential R packages
install.packages(c("ggplot2", "cowplot", "ggpubr", "viridis"))
```

- [ggplot2](https://ggplot2.tidyverse.org) - Grammar of graphics for R
- [cowplot](https://wilkelab.org/cowplot/) - Publication-ready plots
- [ggpubr](https://rpkgs.datanovia.com/ggpubr/) - Publication-ready plots wrapper

### Other Tools

- [GraphPad Prism](https://www.graphpad.com) - Statistical analysis & graphing
- [Origin](https://www.originlab.com) - Data analysis & graphing
- [Tableau](https://www.tableau.com) - Business intelligence visualization
